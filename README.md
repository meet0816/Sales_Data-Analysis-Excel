# 📊 Sales Data Analysis using Excel

## Project Overview
This project analyzes sales data using Microsoft Excel to identify business trends, customer behavior, and profitability.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting
- Slicers
- Formulas

## Objectives
- Analyze total sales
- Calculate profit
- Identify top-selling products
- Compare regional performance
- Analyze monthly sales trends

## Dashboard Features
- Total Sales
- Total Profit
- Sales by Region
- Sales by Category
- Monthly Sales Trend
- Top Products
- Interactive Slicers

## Key Insights
- Highest sales region
- Best performing category
- Most profitable product
- Monthly growth trend
